1. Introduction

When you look up a word in a CD-ROM book using EB Library application, you might wish the application stops displaying book's text at the point where explanation of the word ends. Howerver, EB Library cannot ensure it, since CD-ROM book has no marker in its text which indicates end of paragraph.

Fortunately, many CD-ROM books have a text stop code which could be used for substitute for end of paragraph. EB Library guesses text stop code automatically by default, but sometimes takes a false code. In that case, you have to tell EB Library a correct stop code using Appendix.

The ebstopcode command is a utility to analyze text stop code in a CD-ROM book.

This is edition 4.0, for EB Library version 4.0.
