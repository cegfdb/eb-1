2. Invoke ebstopcode

Its usual way to invoke ebstopcode is as follows:

 
% ebstopcode book-path subbook

book-path points to the top directory of a CD-ROM book where the file `catalog' or `catalogs' resides. It may be a remote access identifer (`ebnet://...'). If book-path is omitted, the current working directory is assumed.

subbook is a directory name of the target subbook. Use the ebinfo command to see what subbooks your CD-ROM has:

 
% ebinfo /cdrom
disc type: EB/EBG/EBXA/EBXA-C/S-EBXA
character code: JIS X 0208
the number of subbooks: 2

subbook 1:
  title: New English Japanese Dictionary (4th edition)
  directory: english
  search methods: word endword keyword menu
  font sizes: 16 24 30 48
  narrow font characters: 0xa121 -- 0xa24e
  wide font characters: 0xa321 -- 0xa27e

subbook 2:
  title: CD-ROM French Japanese Dictionary (3rd edition)
  directory: french
  search methods: word endword keyword menu
  font sizes: 16 24 30 48
  narrow font characters: 
  wide font characters: 0xa321 -- 0xa27e

(See section `Invoke ebinfo' in ebinfo, for more details about the ebinfo command).
In this example, there are two subbooks in the book, and their directory names are english and french.

Type the follwing to analyze text stop code of the english subbook:

 
% ebstopcode /cdrom english

ebstopcode outputs text and candidates for stop code of the subbook, like this:

 
=== stop-code?: 0x1f09 0x0001 ===

=== stop-code?: 0x1f41 0x0100 ===
A

=== stop-code?: 0x1f09 0x0001 ===
n 1: the blood group whose red cells carry the A antigen.

=== stop-code?: 0x1f09 0x0001 ===
2: a metric unit of length equal to one ten billionth of a meter.

=== stop-code?: 0x1f09 0x0001 ===
3: the basic unit of electric current adopted under the System
International d'Unites.

=== stop-code?: 0x1f09 0x0001 ===

=== stop-code?: 0x1f41 0x0100 ===
(snip)

Each `stop-code?:' line shows a candidate for stop code. Specifying it as text stop code, EB Library stops displaying text at the point.

The appropriate stop code of the english subbook is "0x1f41 0x0100". Test the stop code using `--no-candidate' and `--code' options:

 
% ebstopcode --no-candidate --code "0x1f41 0x0100" /cdrom english
A
n 1: the blood group whose red cells carry the A antigen.
2: a metric unit of length equal to one ten billionth of a meter.
3: the basic unit of electric current adopted under the System
International d'Unites.
