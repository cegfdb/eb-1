関連するソフトウェア

EB4J

EB4J は Java で実装した EB ライブラリです。

公式 WWW ホームページ:
https://sourceforge.jp/projects/eb4j/

海老 (EBI)

海老 (EBI) は BTRON 仕様 OS 超漢字 上で動作する電子辞書検索アプリケーションです。 海老は、CD-ROM 書籍へのアクセスに EB ライブラリを使用しています。

公式 WWW ホームページ:
http://hp.vector.co.jp/authors/VA024250/btron/ebi/

eblook

eblook は、対話型の CD-ROM 書籍検索ユーティリティです。 eblook は、CD-ROM 書籍へのアクセスに EB ライブラリを使用しています。

公式 WWW ホームページ:
http://openlab.ring.gr.jp/edict/eblook/

ebmodule

ebmodule は EB ライブラリを Python で利用するための拡張モジュールです。

公式 WWW ホームページ:
http://www.asahi-net.or.jp/~rd6t-kjym/python/index-ja.html#ebmodule

EBook Viewer

EBook Viewer は BeOS 上で動作する電子辞書検索アプリケーションです。 EBook Viewer は、CD-ROM 書籍へのアクセスに EB ライブラリを使用していま す。

公式 WWW ホームページ:
http://home.e07.itscom.net/ytworld/BeWare.html

EBView

EBView は電子辞書検索のための GTK+ のアプリケーションです。 EBView は CD-ROM 書籍へのアクセスに EB ライブラリを使用しています。

公式 WWW ホームページ:
http://ebview.sourceforge.net/

EPWUTIL

EPWUTIL は電子ブックや EPWING の CD-ROM 内のファイルを改変するユーティ リティーを集めたものです。

公式 WWW ホームページ:
http://openlab.ring.gr.jp/edict/epwutil/

FreePWING

FreePWING は書籍データを JIS X 4081 (EPWING のサブセット) へ整形する ソフトウェアです。 EB ライブラリは、FreePWING で整形した書籍データも扱うことができます。

公式 WWW ホームページ:
http://www.sra.co.jp/people/m-kasahr/freepwing/
公式の書籍データサイト (FPWBOOK):
http://openlab.ring.gr.jp/edict/fpw/

gxdic

公式 WWW ホームページ:
http://members.jcom.home.ne.jp/satorut/

KEBook

http://www.os.rim.or.jp/~s_mitu/library/mysofts.html#kebook

xyaku

xyaku は英和・和英の翻訳ツールで、UNIX 系 OS 上で動作します。 add-in モジュール `searcheb.rb' を使用することで、xyaku は CD-ROM 書籍 を引くことがきます。 この add-in モジュールは、RubyEB を使用してい ます。

公式 WWW ホームページ:
http://home.catv.ne.jp/pp/ginoue/software/xyaku/
