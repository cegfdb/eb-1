セキュリティについての注意

zlib 1.1.3 およびそれ以前のもの には、セキュリティに関する不具合が発見されています。 EB ライブラリは zlib を使用していますので、この不具合は EB ライブラリ にも影響します。 また、EB ライブラリ 3.2 およびそれ以前のバージョンの配布ソースコード には、この不具合を持った zlib のソースコードを同梱しています。

EB ライブラリ 3.2.1 からは、配布ソースコードに修正した zlib のソース コードを同梱しています。

EB ライブラリ 3.2 およびそれ以前のものを (お薦めはしませんが) コンパ イルしてインストールしたいときは、 configure に必ず --with-included-zlib=no オプションを指定して、同梱の zlib は使わないようにして下さい。

zlib の不具合に関して詳しくは、次を参照して下さい。

  • Zlib Advisory 2002-03-11
  • CERT Advisory CA-2002-07 Double Free Bug in zlib Compression Library
